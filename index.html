Express.js

¿Qué es Express.js?

Express.js es un framework para crear aplicaciones web Es software libre y de código abierto con licencia MIT.

Ventajas

  • Es minimalista, lo que lo hace muy liviano ya que muchas de sus funcionalidades

  • Maneja Routing, con wildcards, parámetros, etc.

  • Permite la instalación de plugins a travpes de middlewares.

Usos

  • Paginas web.
  • Robusto para futuras versiones para versiones moviles.
